# InstantFood

## ⚙️ Getting Started

To run the project locally:

1. **Clone or Download** the repository.
2. **Unzip** the file if downloaded as a ZIP.
3. **Open `index.html`** in any modern web browser.

No server or backend required.

## 📦 Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
